## 🚀 Projects

### StaySphere

StaySphere is a full-stack property listing web application inspired by platforms like Airbnb.

The goal of this project was to practice real backend concepts such as authentication, database relationships, image uploads, and location-based features.

**Live Project**
https://staysphere-4ea7.onrender.com/

**Key Features**

* User authentication and authorization
* Create, edit and delete property listings
* Image upload support
* Reviews system
* Location integration using Mapbox
* Server-side validation

**Tech Stack**

Node.js
Express.js
MongoDB
EJS
Bootstrap
Mapbox
